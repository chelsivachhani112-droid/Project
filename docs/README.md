## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Node.js (for local development)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/finova.git
   cd finova
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```
